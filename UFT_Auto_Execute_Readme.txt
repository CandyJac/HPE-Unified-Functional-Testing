PowerShell Script Steps for UFT Automatic Test Case Execution

#1 : Kill Specified Processes

#2 : Delete Windows Temporary and User Temporary Files

#3 : Clear Internet Explorer Temporary Files and Cookies

#4 : Start UFT and Execute the Test Case

	* Load the Test Case in UFT
	* Wait for the Test to Execute and add log entry to Enevt Viewer
	* Once Test completed FlightGUI and UFT closes and log an entry to Event Viewer
